## 🚀 Deployment

### GitHub Pages
1. Repository: `Black-Dog-Official-Music/black-dog-official-music`
2. Branch main contiene il codice
3. Per abilitare GitHub Pages:
   - Vai in Settings > Pages
   - Seleziona "Deploy from a branch"
   - Seleziona "main" come source

### URL del Sito
```
https://Black-Dog-Official-Music.github.io/black-dog-official-music/
```

### Setup locale
```bash
git clone https://github.com/Black-Dog-Official-Music/black-dog-official-music.git
cd black-dog-official-music
# Aprire index.html nel browser
```
